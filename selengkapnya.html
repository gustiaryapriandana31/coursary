Video 1 -- Pendahuluan Mengenai HTML

Diupload 1 minggu yang lalu oleh Tim Web Development Coursary

HTML atau Hypertext Markup Language merupakan bahasa markup yang digunakan untuk membangun konten-konten dari sebuah website yang akan kita buat. HTML ini bukanlah sebuah bahasa pemrograman, melainkan sebuah bahasa markup yang digunakan untuk membuat perintah menampilkan konten suatu halaman web melalui web browser.

Dalam penerapannya, kita akan mempelajari mengenai macam-macam tag pada HTML yang fungsinya untuk menampilkan ke halaman web kita. Perlu diketahui juga bahwa HTML hanya untuk membuat struktur dari suatu website saja dan belum dengan stylenya. Oleh karena itu, kepolosan dari suatu web yang dibuat merupakan salah satu citi dari membuat website dengan HTML ini.
