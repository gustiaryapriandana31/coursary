Apa yang Diperlukan untuk Membuat Website

Untuk menjadi seorang web developer,anda harus memulai dengan mempelajari dasar-dasar pemrograman web. Anda membutuhkan teknologi HTML, CSS, JavaScript, dan Framework-framework untuk membangun website sendiri

Bagaimana Cara Membuat Website

Membuat website bisa anda lakukan dengan beberapa pilihan. Anda bisa men- develop -nya sendiri dengan mengoding dari awal. Anda juga bisa menggunakan template-template website yang sudah banyak dipublikasikan. Semua tergantung kebutuhan.

Apa yang Ditawarkan di Website ini

Disini anda akan mendapatkan materi-materi dasar-dasar pemrograman web, mulai dari teknologi HTML, CSS, JavaScript, bahkan sampai Framework-frameworknya. Anda bisa membaca dan menonton materi-materi yang sudah kamu cantumkan di dalam website pembelajaran ini.

Highlight Perkembangan Teknologi CSS

Tailwind css adalah css framework yang menawarkan konsep baru dengan konsep utility first di mana kamu tidak perlu lagi memikirkan panjang nama kelas untuk komponen HTMLnya. Tailwind terdiri dari nama kelas yang cukup intuitif untuk digunakan.

Tailwind CSS adalah kerangka kerja CSS yang mengutamakan utilitas. Artinya, alih-alih menulis banyak CSS, Anda akan menulis banyak kelas untuk elemen HTML. Di Tailwind CSS, Anda memiliki kelas untuk hampir semua jenis margin, padding, latar belakang, ukuran font, jenis font, warna teks, bayangan, dan sebagainya.

Begitu pula untuk membuat desain yang responsif menggunakan tailwindcss ini sangatlah mudah, tailwindcss menyediakan lima breakpoints yang dapat kita gunakan bersamaan dengan utility class.
